# dynamic-blog-website

Hello, and welcome to this GitHub repo, which has eventually become a (somewhat) usable (locally-saved) blog!

Homepage:
> This will display a list of blog posts created and saved locally, along with images if added

Create:
> This will allow for the creation of a rudimentary blog post along with a title, text body and an image

Edit/Delete:
> This will allow for the edit and deletion of locally saved blog posts
